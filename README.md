## 安装和编译

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## 使用技术
vue2 + vuex + vue-router + vue-resource + ES6 + element-ui + sass
